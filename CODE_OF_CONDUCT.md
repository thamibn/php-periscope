# Code of Conduct

This project follows the **[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**. By participating in this project — opening issues, submitting PRs, joining the Discord, or interacting in any other community channel — you agree to abide by its terms.

The full text lives at the link above; we adopt it verbatim rather than mirroring it here so we always reflect the canonical, up-to-date version.

## Reporting

Concerns should be reported privately to the project maintainer via email (see `composer.json` `lead` block or the repo's `Owners`). Reports stay confidential. We respond within 7 days.
